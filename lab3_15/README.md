## Мета роботи

Ознайомитись з принципами **спадкування**, **виклику конструкторів базового класу**, **перевизначення методів (virtual/override)** та **реалізації поліморфізму** у C#.

---

## Опис виконання завдання

Було створено базовий клас **`Robot`**, який має віртуальний метод `Work()`.  
Від нього успадковано два похідні класи:

- **`CleanerRobot`** — реалізує метод `Work()` для моделювання роботи прибирального робота.
- **`DeliveryRobot`** — реалізує метод `Work()` для доставки посилок.

Для демонстрації **поліморфізму** створено колекцію `List<Robot>`, яка містить об'єкти обох типів.  
При виклику `Work()` для кожного елемента списку відбувається **динамічний виклик** відповідної реалізації.
